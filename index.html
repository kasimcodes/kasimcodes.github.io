Yeteneklerim

Back-end Development

C# ve .NET ile Temel Geliştirme: Microsoft .NET çatısı üzerinde C# dilini kullanarak web uygulamaları ve servisleri geliştirme yolunda ilerliyorum. Temel C# OOP prensiplerini ve .NET Core yapısını anlamış durumdayım. Basit düzeyde RESTful API mantığını kavramış olup, staj projelerinde sunucu tarafı mantığı kurma ve test etme deneyimi kazandım. Güvenli ve temiz kod yazma pratiklerini öğrenmeye odaklanıyorum.

Front-end Development

HTML, CSS ve JavaScript Temelleri: Modern ve duyarlı (responsive) web arayüzleri oluşturmak için HTML ve CSS'e hakimim. Kullanıcı etkileşimini artırmak amacıyla temel JavaScript fonksiyonlarını kullanabiliyorum. Kullanılabilirlik ve erişilebilirlik konularına dikkat ederek, kullanıcı odaklı tasarımları koda dönüştürme konusunda aktif olarak pratik yapıyorum.

Full Stack Development

Bütünleşik Proje Geliştirme: Projelerde hem ön yüz (Front-end) hem de arka yüz (Back-end) teknolojilerini birleştirme becerisi kazanıyorum. Özellikle .NET tabanlı uygulamalarda veri akışını yönetme ve uçtan uca işlevsellik sağlama pratiği yapıyorum. İki taraf arasındaki bağlantıyı kurarak bir uygulamanın nasıl çalıştığını bütünsel olarak görme ve temel entegrasyon süreçlerini anlama yeteneğindeyim.

Computer Programing

Veritabanı ve Algoritmik Temel: Programlama mantığı ve temel veri yapıları üzerine çalışıyorum. C# ile problem çözme ve algoritma geliştirme becerilerimi ilerletiyorum. SQL kullanarak veritabanı sorguları yazabiliyor ve uygulamaların ihtiyaç duyduğu veriyi doğru şekilde çekip işleyebiliyorum. Sürüm kontrol sistemi Git ile temel takım çalışması pratiklerine aşinayım.
